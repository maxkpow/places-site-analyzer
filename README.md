# Scany
Tool uses selenium to analyze website for POI (Points of interest) data existance.
UML https://app.diagrams.net/#G1hv3iaBeUxmB34LA99gm7ypodzCKKft-x

Tool extracts:
- http requests
- ul & ol list tags
- links from a tags
- scripts

#### Installing
    pip install --editable .


#### Usage
    scany run --url https://avoska.ru
